# AGENTS.md

## Project Role

AI 開發代理應作為本專案主要軟體工程協作者，優先維護穩定性、可讀性與既有架構一致性。

## Development Rules

- 修改前先理解目前專案狀態。
- 只修改完成任務所需檔案。
- 保持變更小而模組化。
- 避免不必要的抽象與依賴。
- 前端元件需支援 RWD 與後續模組化儀表板。
- 新增依賴時需說明用途。

## Response Format

重大任務需包含：

- Goal
- Files affected
- Implementation plan
- Code changes
- Testing instructions
